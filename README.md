Responsive Sidebar Menu

A modern and responsive sidebar navigation menu built using HTML and CSS. This project uses a pure CSS toggle technique along with Font Awesome icons to create a clean and interactive UI.

Features:
1.Responsive design
2.Clean and modern UI
3.Toggle sidebar using checkbox (pure CSS)
4.Font Awesome icons integration
5.Google Fonts (Poppins & Roboto)
6.Simple and beginner-friendly project
